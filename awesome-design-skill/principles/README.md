# Design Principles

本目录包含设计原则和最佳实践，作为所有设计风格的基础指导。

## 原则文件

| 文件 | 类型 | 描述 |
|------|------|------|
| [general-design.md](./general-design.md) | 理论指导 | 10 个通用设计原则，适用于所有设计任务 |
| [refactoring-ui.md](./refactoring-ui.md) | 实战技巧 | 8 个 UI 重构原则，适用于重构现有 UI |

## 使用说明

这两个原则文件会被 AI 自动加载，作为设计风格的补充。当应用任何设计风格时：

1. **风格特定原则优先**：风格 prompt.md 中明确指定的规则
2. **通用原则回退**：风格未提及的方面，使用本目录中的通用原则
